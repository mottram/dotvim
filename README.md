### Setup

1. `git clone https://github.com/mottram/dotvim.git ~/.vim`
2. `mkdir -p ~/.vim/{backup,temp,undo,view}`
2. `ln -s ~/.vim/vimrc ~/.vimrc`
3. `git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle`
4. `vim +BundleInstall +qall`
